Requirements

  • WordPress 2.2 or 2.0.x/2.1.x with WordPress Widgets

Installation

  1. Download get-in-touch-plugin.zip
  2. Extract and upload get-in-touch.zip to the wp-content/plugins/ directory.
  3. Enable Get in Touch in the Plugin admin panel.
  4. Go to the widget admin panel ( Appearance > Widgets ), drag the widget to the sidebar and then customize options.
